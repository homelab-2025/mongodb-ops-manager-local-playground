# Ops Manager Playground

This repository provides a simple way to set up a local playground environment for MongoDB Ops Manager using Docker Compose or WSL2 with Ubuntu 24.04.

## Contents

- [WSL2 Setup](wsl2/README.md): Instructions to set up MongoDB Ops Manager on Ubuntu 24.04 running on WSL2.
- [Docker Compose Setup](docker-compose/README.md): Instructions to set up MongoDB Ops Manager using Docker Compose.
